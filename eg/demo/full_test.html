Fuller Demonstration of Capabilities

This page tries to test as many capabilities of the Tk::HyperText's as possible, specifically testing edge cases. In some ways, HTML is a very simple language, however there are a number of interesting combinations and edge cases, so try them here to test that Tk::HyperText does the right thing (as much as possible).
